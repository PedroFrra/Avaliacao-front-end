## Descrição do Projeto
NexusHub é uma aplicação front-end desenvolvida em React com Vite que representa um painel moderno de serviços digitais. A plataforma apresenta uma interface responsiva e componentizada para exibição de serviços tecnológicos de uma empresa fictícia.

## Tecnologias Utilizadas
- **React 18** - Biblioteca JavaScript para construção de interfaces
- **Vite** - Build tool moderna e rápida
- **React Bootstrap** - Framework CSS para componentes responsivos
- **Bootstrap 5** - Sistema de grid e estilização
- **Lucide React** - Biblioteca de ícones modernos
- **CSS3** - Estilização personalizada

## Design System
O projeto utiliza uma combinação de:
- Bootstrap para grid system e componentes base
- Paleta de cores personalizada com tons de azul e roxo
- Tipografia Inter para melhor legibilidade
- Ícones Lucide React para consistência visual
- Animações suaves para melhor experiência do usuário
